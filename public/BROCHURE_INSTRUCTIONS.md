Place the brochure PDF in the project so the site serves it as a static file.

1. Name your brochure file `brochure.pdf`.
2. Copy or paste `brochure.pdf` into the `public/` folder of the project root.

After adding the file, the Download Brochure button on the Partnerships page will link to `/brochure.pdf` and allow users to download it.

If you run the dev server (Vite), the change is immediately available. For production, rebuild/deploy as usual.
